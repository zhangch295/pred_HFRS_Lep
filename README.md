# pred_HFRS_Lep

**pred_HFRS_Lep** is a research-oriented project for predicting the temporal dynamics of **Hemorrhagic fever with renal syndrome (HFRS)** and **Leptospirosis** using multi-source data.

---

## 🧩 Overview

Hemorrhagic fever with renal syndrome (HFRS) and Leptospirosis are important zoonotic infectious diseases with clear temporal and seasonal patterns. Their occurrence is influenced by climate, ecological environment, population-related factors and socioeconomic conditions.

This project aims to develop a multi-source data-driven framework for analyzing and predicting monthly disease incidence, supporting disease surveillance, risk assessment and early warning.

---

## 📂 Data Sources

The study uses monthly disease surveillance data and related external predictors from **January 2010 to December 2025**.

The main data sources include:

- Monthly reported cases of **HFRS**
- Monthly reported cases of **Leptospirosis**
- Meteorological data
- Environmental and ecological indicators
- Remote sensing-derived indicators
- Socioeconomic indicators
- Population-related variables
- Temporal and seasonal features

Due to data-sharing restrictions, raw surveillance data are not publicly distributed in this repository.

---

## ⚙️ Methods

The main methods used in this project include:

- Time-series data preprocessing
- Multi-source feature integration
- Lagged feature construction
- Seasonal and trend analysis
- Time-series decomposition
- Machine learning-based prediction models
- Model performance evaluation
- Feature importance and model interpretation

The modelling framework focuses on comparing prediction performance across different feature sets and identifying key factors associated with the temporal dynamics of HFRS and Leptospirosis.

---

## 📌 Research Significance

This project provides a reproducible analytical framework for predicting zoonotic infectious diseases using multi-source data. By integrating disease surveillance, climate, environmental and socioeconomic information, the framework may help improve understanding of disease dynamics and provide methodological support for early warning and targeted public health interventions.
